M2-Team

M2-Team is originally a software development team created by Himi Misaki, Kenji Mouri and Ruqing Yang for the development of NSudo project in June 18, 2015. Today, M2-Team is more like friend circle and personal team for Kenji Mouri due to Himi Misaki and Ruqing Yang had faded out of the team since at least 2018.


Active Projects

  • NanaZip - The 7-Zip derivative intended for the modern Windows experience
  • NanaBox - The third-party lightweight XAML-based out-of-box-experience oriented Hyper-V virtualization software based on Host Compute System API, Remote Desktop ActiveX control and XAML Islands
  • NanaGet - Lightweight file transfer utility based on aria2 and XAML Islands
  • NanaRun - Application runtime environment customization utility
  • Privexec - Run the program with the specified permission level (C++20 required)
    See https://github.com/M2Team for all projects which are actively maintained by M2-Team.

Deprecated Projects

  • NSudo - Series of System Administration Tools (Superseded by NanaRun)
  • NWSRMgr - Windows System Restore Manager (Deprecated without alternatives created by M2-Team)
  • Nagisa - An open source file transfer utility on Universal Windows Platform (Superseded by NanaGet)
    See https://github.com/M2TeamArchived for all projects which are archived by M2-Team.
